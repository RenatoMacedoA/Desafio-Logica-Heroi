## 1️⃣ Desafio: Classificador de Nível de Herói

### O que deve ser utilizado
- Variáveis  
- Operadores  
- Laços de repetição  
- Estruturas de decisão  

### Objetivo
Criar uma variável para armazenar o **nome** e a **quantidade de experiência (XP)** de um herói, depois utilizar uma **estrutura de decisão** para determinar o nível:

| XP do Herói       | Nível       |
|------------------|------------|
| < 1.000          | Ferro      |
| 1.001 a 2.000    | Bronze     |
| 2.001 a 5.000    | Prata      |
| 5.001 a 7.000    | Ouro       |
| 7.001 a 8.000    | Platina    |
| 8.001 a 9.000    | Ascendente |
| 9.001 a 10.000   | Imortal    |
| ≥ 10.001         | Radiante   |

---

### Saída Esperada
Ao final, a aplicação deve exibir a seguinte mensagem:

O Herói de nome {nome} está no nível de {nivel}



---

### Tecnologias
- JavaScript
- HTML (opcional)
- CSS (opcional)

---

### Como usar
1. Clone ou faça o download deste repositório.
2. Abra o arquivo `index.html` ou execute o script `heroi.js` no seu ambiente preferido.
3. Insira o nome do herói e a quantidade de XP.
4. Veja a mensagem com o nível do herói exibida na tela.
